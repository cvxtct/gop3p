# GOP3P

**The useless project to keep my gappy Go in shape.**

It fixes Python PEP3 `trailing-whitespace` and `missing-final-newline` on files in a given folder using channels and goroutine.

### Install

`make all`

### Usage

`gop3p -dir /your/python/project/`
